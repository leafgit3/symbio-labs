# Session Workflow (Cycle-by-Cycle)

Use one report file per session and fill it gradually.

## Naming
`YYYY-MM-DD__session_<short-id>__<scenario-label>.md`

Example:
`2026-03-16__session_2a6345af__control_baseline_replay.md`

## Recommended Loop
1. Create file from `../templates/scenario_run_log.md`.
2. Run cycle 1 and fill only Cycle 1 block.
3. Repeat per cycle.
4. Complete scenario wrap-up.
5. Copy key findings into session-level robustness report.

## Pace Guidance
- Slow is fine.
- Prioritize accurate capture over speed.
- Write interpretations only after there is enough evidence.
